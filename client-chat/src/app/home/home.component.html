The web framework Actix

Actix-web is a powerful and pragmatic framework. The server purpose is double here.

The first purpose is to serve the SPA front-end application.

  • For this the server use the actix_files crate to handle the statics files in the Angular SPA dist directories.
  • With in one hand the actix_files::Files struct.
  • In the other hand the actix_files::Named_Files struct is used to served the index.html file which is the Angular SPA root file.

    • The second purpose is to run a Web Socket server.

  • This server used by the Angular SPA to handle the chat live messaging logic.
  • The Chat Server Actor (actix-web-actors) is a System Service that runs in the same thread as the HttpServer/WS Listener.
  • The implementation has been followed the example of jtara1.

    • A Angluar SPA to render the all

    The web-platform Angular v.12

    Angular is a platform for building mobile and desktop web applications.

      The front-end handle two mains application mechanics :

    • The first one is the application front-end routing. The Angular routing allows to separate the application in component and easily associate them to url.
    • Then the rxjs/webSocket handle the client side of the WebSocket Server. The WebSocketSubject constructor allows to open a link with the webSocket server and create a session while parameting the data transfered.
